## Tungsten

1. Install tungsten (e.g. `yay tungsten`)
2. Get API key from http://developer.wolframalpha.com/portal/myapps/
3. Put it in `~/.wolfram_api_key`
4. Use `~/bin/rofi_run -t` or `~/bin/tungsten` directly
